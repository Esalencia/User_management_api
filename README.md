# User_management_api
User management crud api a simple REST API that allows a frontend application to manage users.

1. The application does the following
• Returns a list of users
• Each user should contain: id, name, email
• Gets a single user by id
• deletes a single user by id
• Creates a new user
• Creates users in bulk
2. The data is stored in the following 
• Store users in memory (array, list, or map)
• IDs should are  auto-incremented   
3. Error Handling
• Handle invalid requests gracefully
• Return meaningful error messages
